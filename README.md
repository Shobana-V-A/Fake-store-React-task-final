# Fake Store Cart App

A responsive ReactJS application that fetches product data from the Fake Store API and allows users to add items to a shopping cart.

## Features
- Fetches and displays products from `https://fakestoreapi.com/`.
- Responsive grid layout using Tailwind CSS.
- Users can add items to their cart.
- Alerts the user if an item is already in the cart.
- Real-time cart counter in the navigation bar.
- Modal interface to view cart items.
- Ability to remove items directly from the cart modal.

## Tech Stack
- **ReactJS** (Hooks: `useState`, `useEffect`)
- **Tailwind CSS** (for styling and responsiveness)
- **JavaScript** (ES6+ syntax)

## Setup Instructions
1. Clone the repository or download the files.
2. Run `npm install` to install dependencies (React, Tailwind).
3. Run `npm run dev` (if using Vite) or `npm start` (if using Create React App) to start the local development server.
